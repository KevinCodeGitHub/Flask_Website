About

Our project is an exciting journey in developing an On-Board Diagnostics 2 (OBD2) system from scratch. With a passion for automotive technology, we've embarked on this endeavor to create a robust OBD2 solution that provides valuable insights into your vehicle's performance.

Key features

Ground-Up Development
Our OBD2 system is built from the ground up, showcasing our commitment to innovation and excellence in automotive diagnostics.


MCP2515 CAN-Controller
Utilizing the perks of the MCP2515, we ensure accurate and reliable communication with the vehicle's onboard system.


MCP2551 CAN-Transceiver
The MCP2551 is a High-Speed CAN-Transceiver that serves as the interface between a CAN protocol controller and the pysical bys. This component is fully compatible with the ISO-11898 standard.
